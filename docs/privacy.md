# Privacy Policy

## Data Collected
- Phone number (masked in dashboard views)
- Caller transcript
- Ticket metadata (category, severity, ward, status, timestamps, evidence URL)

## Retention
- Data is retained for 90 days by default for grievance lifecycle and audit workflows.

## Third-Party Sharing
- No personally identifiable information is shared with third parties except required platform processors used for operation (telephony and email infrastructure).

## Consent Mechanism
- Consent is captured through IVR opt-in before grievance intake starts.

## Deletion Requests
- Citizens can request deletion by contacting: privacy@jansamvaad.example
